# Graphics Renderer UI - Windows Quick Start

## Quick Build & Run

1. **Install Prerequisites:**
   - Visual Studio 2022 with C++ tools
   - CMake 3.18+
   - .NET 8.0 SDK
   - CUDA Toolkit (optional, for ray tracing)

2. **Build:**
   ```powershell
   .\build-ui.ps1
   ```

3. **Run:**
   ```powershell
   cd GraphicsRendererUI
   dotnet run -c Release
   ```

## Features

- ✅ Cross-platform (Windows, Linux, macOS)
- ✅ Ray Tracing (CUDA) and Rasterization modes
- ✅ OBJ file import support
- ✅ Real-time transform editing
- ✅ Interactive GUI with Avalonia
